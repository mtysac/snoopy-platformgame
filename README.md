# Snoopy!!!! — 2D Platformer

A 2D side-scrolling platformer built with Python and pygame. Includes a main menu, gameplay loop, and a tile-based level editor.

## Requirements

- Python 3.x
- pygame

Install pygame with:
```
pip install pygame
```

## Running the Game

**With main menu:**
```
python main.py
```

**Straight into gameplay (no menu):**
```
python game.py
```

**Level editor:**
```
cd scripts
python editor.py
```

## Controls

### Gameplay
| Key | Action |
|-----|--------|
| A | Move left |
| D | Move right |
| W | Jump |
| ESC | Return to main menu |

### Level Editor
| Key / Input | Action |
|-------------|--------|
| A / D / W / S | Pan camera |
| Scroll wheel | Cycle tile type |
| Shift + Scroll | Cycle tile variant |
| Left click | Place tile |
| Right click | Remove tile |
| G | Toggle on-grid / off-grid placement |
| T | Auto-tile |
| O | Save map to `map.json` |

## Project Structure

```
├── main.py              # Entry point with main menu
├── game.py              # Standalone gameplay (no menu)
├── map.json             # Level data (generated by editor)
├── scripts/
│   ├── entities.py      # Player and physics entity logic
│   ├── tilemap.py       # Tilemap rendering, collision, save/load
│   ├── clouds.py        # Background cloud system
│   ├── editor.py        # Level editor
│   ├── button.py        # UI button component
│   └── utils.py         # Image/animation loading helpers
├── data/
│   ├── images/          # Sprites (player, tiles, food, clouds)
│   └── *.mp3            # Music tracks
└── assets/              # Menu fonts and UI images
```

## Tile Types

- **grass** — physics-enabled ground tile with autotiling support
- **cement** — physics-enabled platform tile
- **food** — decorative off-grid collectible sprites
- **back** — background decoration tiles
